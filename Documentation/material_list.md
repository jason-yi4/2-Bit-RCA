# Materials and Equipment for RCA

| ITEM | AMOUNT | DESCRIPTION |
| :--: | :----: | :---------: |
| Breadboard | 1 | Used as the main board to connect all circuit components. |
| Battery Box | 1 | Used in place of a DC power supply, 5V. |
| Wire | n/a | Used as the physical connections between components. |
| 8-Position DIP Switch Array | 1 | Used to simulate bit inputs. |
| 10-Segment LED Bar | 1 | Used as a visual indicator of carry and sum bits. |
| 9-Pin 1 k $\Omega$ Resistor Network | 1 | Used to reduce current flow into the DIP Switch Array. |
| 11-Pin 330 $\Omega$ Resistor Network | 1 | Used to further reduce current flow into the LED Bar. |
| 74HC08 Quad 2-Input AND Gate | 1 | Used for carry bit computation. |
| 74HC32 Quad 2-Input OR Gate | 1 | Used for carry bit computation. |
| 74HC86 Quad 2-Input XOR Gate | 1 | Used for sum bit computation. |
